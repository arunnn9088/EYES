## 📖 Description

**EYES** is a behavioral intelligence platform landing page that showcases a system for understanding your digital patterns over time. Features a sleek, dark-themed UI with animated sections for Features, How it Works, Trust & Privacy, and an Early Access waitlist with Google OAuth integration.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/arunnn9088/EYES.git
cd EYES

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## ✨ Features

- **Behavioral pattern tracking** — Connect accounts and see patterns emerge by day 22
- **EU-hosted infrastructure** — Supabase in Frankfurt, fully GDPR compliant
- **Per-connector control** — Pause, exclude, and manage data sources individually
- **Full delete** — One-click account and data deletion
- **Early Access waitlist** — Email signup + Google OAuth integration
